CityGo

CityGo.

CityGo is a Location-Based Mobile Game for IOS and created using Unity3D. The game enviroment is real time enviroment which uses Mapbox SDK to generate the true world location and maps over the world. Within the game there is Mini Games - Snake Games and collectable points. This is build to encourage players to walk around to release anxiety and also stress to help with mental stress. The main purpose of CityGO is to presure players that games not only can help in reducing stress, it helps also in physically. The game is Playtesting with deviecs. If you wish to have a look MUST download the build setting of Unity (IOS/ Andriod).
